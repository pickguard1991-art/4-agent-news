### 🔍 論理的整合性の問題

**「ETF資金流出」と「機関投資家の慎重姿勢」の説明不足**
```

```
> この矛盾を分析が素通りしている。「どのETFから」「どの期間で」「どの程度の規模で」流出しているのかデータが皆無。

---

**「取引量19.30%急増＝底値圏の関心」という解釈の恣意性**

取引量増加は以下のどれでも起こりうる：

| 解釈 | 説明 |
|------|------|
| 底値買い | 分析が採用した解釈 |
| **狼狽売り** | 下落局面では売り増しも取引量増加をもたらす |
| **デリバティブ清算** | 強制決済による出来高増加 |
| **ウォッシュトレード** | 取引所による水増し |

**根拠なしに最も都合の良い解釈を選択している。**

---

**「ビットコインドミナンス上昇＝強気」という単純化**

ドミナンス上昇は以下の文脈でも発生する：
- アルト全体が暴落し、相対的にBTCが下がりにくいだけ
- **市場全体の縮小局面でのリスクオフ**（ポジティブではない可能性）
